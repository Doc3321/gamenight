# משחק המילים - Word Game

משחק מילים אינטראקטיבי בעברית עם תמיכה ב-RTL.

## תכונות

- 🎯 3 סיבובים לכל משחק
- 🎲 3 בחירות בכל סיבוב
- 🎭 מערכת מתחזה - מילה אחת מופיעה פעמיים
- 📚 נושאים שונים לבחירה
- ✨ אנימציות חלקות
- 🌐 תמיכה מלאה בעברית ו-RTL

## טכנולוגיות

- Next.js 15
- TypeScript
- Tailwind CSS
- shadcn/ui
- Framer Motion
- React

## התקנה

```bash
npm install
npm run dev
```

## איך לשחק

1. בחר נושא מהרשימה
2. לחץ על "התחל משחק חדש"
3. לחץ על "סובב" כדי לקבל 3 מילים
4. גלה אם אתה המתחזה או לא
5. המשך לסיבוב הבא

## מבנה הפרויקט

- `src/app/` - דפי האפליקציה
- `src/components/` - קומפוננטים
- `src/lib/` - לוגיקת המשחק
- `src/data/` - נתוני הנושאים והמילים
